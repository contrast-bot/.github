# Contrast Bot
**A versatile Discord bot built with TypeScript and Discord.js**
<br><br>
## About
Contrast Bot is a powerful, multi-purpose Discord bot designed to enhance your server experience. Built with TypeScript, SQLite and Discord.js, he offers a wide range of features from moderation tools to fun commands and social interactions... [Read more here](https://github.com/contrast-bot/contrast-bot/blob/main/README.md)
